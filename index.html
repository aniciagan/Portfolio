Project: E-commerce Checkout Redesign

**Problem:** High cart abandonment rate on the legacy e-commerce platform.

**Solution:** Led a cross-functional team to conduct user research, gather requirements, and document user stories for a streamlined, mobile-first checkout process. Used Figma for wireframing and conducted A/B testing to validate changes.

**Tools:** Agile, Jira, Figma, A/B Testing, User Stories

Project: CRM Data Migration

**Problem:** Inaccurate and incomplete data in a new CRM system, causing reporting issues.

**Solution:** Analyzed existing data sources, created data mapping diagrams, and wrote SQL scripts to clean, transform, and migrate data to the new system. Developed a data quality dashboard in Power BI to monitor and report on data health post-migration.

**Tools:** SQL, Python, Power BI, Data Mapping

Project: Internal Process Automation

**Problem:** Manual, error-prone approval process for internal project requests.

**Solution:** Mapped the existing process using BPMN diagrams. Developed a new automated workflow using a low-code platform, reducing approval time by 75% and minimizing manual errors. Documented the new process in a detailed FRD.

**Tools:** BPMN, FRD, Low-Code Platforms, Process Automation
